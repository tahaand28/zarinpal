# zarinpal
طراحی کامل وب‌سایت زرین‌پال از صفر با استفاده از HTML، CSS و JavaScript. در این پروژه، تمام بخش‌های اصلی از جمله هدر، بنر، معرفی خدمات، بخش کاربران و فوتر و صفحه‌ی ورود و ثبت نام به‌صورت واکنش‌گرا (Responsive) طراحی شده‌اند. هدف این نمونه‌کار، نمایش مهارت من در طراحی فرانت‌اند، رعایت اصول UI/UX و پیاده‌سازی صفحات مدرن بدون استفاده از فریم‌ورک‌ها بوده است.
